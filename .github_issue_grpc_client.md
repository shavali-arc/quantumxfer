### Terminal Commands
```bash
# List services
GRPC-REFLECT grpc://service.internal:50051

# Call method
GRPC-CALL grpc://service.internal:50051/api.Users/GetUser \
  -d '{"id":123}'

# Server streaming
GRPC-STREAM grpc://service.internal:50051/api.Logs/GetLogs \
  --watch

# List methods
GRPC-METHODS grpc://service.internal:50051
```

## ðŸ“Š Use Cases

### 1. Service Discovery
```bash
$ GRPC-REFLECT grpc://user-service:50051

Services:
  - api.Users
    - GetUser(GetUserRequest) -> User
    - ListUsers(ListUsersRequest) -> stream User
    - CreateUser(CreateUserRequest) -> User
    
  - api.Orders
    - GetOrder(GetOrderRequest) -> Order
    - ListOrders(ListOrdersRequest) -> stream Order
```

### 2. Call Service Method
```bash
$ GRPC-CALL grpc://user-service:50051/api.Users/GetUser \
  -d '{"id":"user_123"}'

{
  "id": "user_123",
  "name": "John Doe",
  "email": "john@example.com",
  "createdAt": "2025-01-15T10:30:00Z"
}

Duration: 45ms
```

### 3. Streaming
```bash
$ GRPC-STREAM grpc://log-service:50051/api.Logs/GetLogs \
  -d '{"service":"api","level":"ERROR"}' \
  --watch

>>> {timestamp: "2025-11-29T14:30:45Z", message: "Database error"}
>>> {timestamp: "2025-11-29T14:30:46Z", message: "Retry attempt 1"}
>>> {timestamp: "2025-11-29T14:30:47Z", message: "Connection recovered"}
```
